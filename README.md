# Pakkapon MVVM-Templates
A file template for Xcode. It creates all necessary files for MVVM architecture.

Installation
------------

To install the Pakkapon MVVM Xcode templates, run:
```
make install
```
To uninstall the Pakkapon MVVM Xcode templates, run:
```
make uninstall
```
Requirements
------------

* Xcode 9.0+
* Swift 5
